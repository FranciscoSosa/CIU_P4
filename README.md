## Descripción
Esta práctica parte de la anterior y se le añade una nave que navega por el sistema planetario, la función de esta nave es hacer 
de cámara para ver distintos puntos del sistema, esto se ha llevado a cabo usando la función camera() de processing y configurando
los distintos tipos de parámetros que esta tiene, además la aplicación permite intercambiar entre la vista genral desde el sistema
y la vista desde la nave.
